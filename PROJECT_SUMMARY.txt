================================================================================
CONFIGURATION REQUIREMENTS
================================================================================
Before Building:
  1. Download google-services.json from Firebase Console
  2. Place it in app/ folder
  3. Configure AdMob IDs in Constants.kt
  4. Update AndroidManifest.xml with AdMob App ID

Before Publishing:
  1. Create Firebase project
  2. Set up Firestore with security rules
  3. Configure Firebase Authentication
  4. Create AdMob account and units
  5. Generate release keystore
  6. Create Google Play Developer account
  7. Prepare app assets and descriptions

QUICK COMMANDS
================================================================================
Build Debug APK:
  ./gradlew assembleDebug

Build Release APK:
  ./gradlew assembleRelease

Install on Device:
  ./gradlew installDebug

Run Tests:
  ./gradlew test

Clean Project:
  ./gradlew clean

View Logs:
  adb logcat

Get SHA-1:
  keytool -list -v -keystore ~/.android/debug.keystore
